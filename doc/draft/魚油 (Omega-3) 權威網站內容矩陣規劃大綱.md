## **3\. Spoke Page (品質面)：濃度與檢驗**

建議標題：魚油濃度 30% 跟 85% 差在哪？揭露低濃度魚油的隱藏風險  
目標關鍵字：高濃度魚油, 魚油檢驗, IFOS認證  
讀者意圖：不想繳智商稅，擔心食安問題。

### **文章大綱**

1. **為什麼專家都推「高濃度」魚油？**  
   * **算術題**：吃 1 顆 80% \= 吃 3 顆 30%。  
   * **關鍵揭露**：低濃度魚油剩下的 70% 是什麼？（飽和脂肪酸、多餘熱量、雜質）。  
   * 身體負擔論述。  
2. **檢驗報告怎麼看？拒絕重金屬與塑化劑**  
   * **IFOS 五星認證**是什麼？為什麼它是最高標準？  
   * 重金屬（汞、鉛、砷）對神經的危害。  
   * 核輻射檢驗（針對日本來源或太平洋漁獲）。  
3. **新鮮度指標 (PV/AV值) 是什麼？**  
   * 為什麼你的魚油有腥味？氧化油的危害比沒吃更可怕。  
4. **教你計算：如何算出真實的 EPA+DHA 含量？**  
   * 手把手教學：看營養標示 (份數 x 每份含量 x 濃度)。

   * 魚油：深海魚萃取（EPA+DHA 均衡）。  
   * 藻油：微藻萃取（DHA 為主，素食可）。  
   * 磷蝦油：南極磷蝦萃取（含蝦紅素，磷脂質型態）。  
2. **生物利用率與吸收大戰**  
   * 磷蝦油的磷脂質結構真的吸收比較好嗎？（解析誇大的行銷話術）。  
   * 濃度硬傷：為什麼磷蝦油很難做到高濃度？  
3. **環保與永續議題**  
   * 磷蝦濫捕與企鵝保育問題。  
   * 藻油的環境友善優勢。  
4. **懶人包：你該選哪種？**  
   * 一般保養/心血管/抗發炎 ➝ 選魚油。  
   * 孕婦/素食/對魚腥味極度敏感 ➝ 選藻油。  
   * 想順便補充蝦紅素/嘗鮮 ➝ 選磷蝦油（但 CP 值通常較低）。
